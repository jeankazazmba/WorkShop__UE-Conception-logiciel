# WS  : Système Hospitalier fait en Java

Mini application Java basée sur un diagramme UML,
permettant la gestion des naissances et des allocations familiales.

## Fonctionnalités
- Authentification
- Gestion des rôles
- Enregistrement des naissances
- Gestion des allocations
